在iPhone手机上，可以使用快捷指令中的SSH指令，将文件上传到树莓派。

首先，利用**“制作归档”**指令，将文件压缩成.tar.gz的压缩文件。

然后，利用**“通过SSH运行脚本”**指令，将压缩文件上传到树莓派，并且解压缩文件。

例如以下命令，就是将文件上传并解压缩到树莓派的**/home/pi**目录：

```bash
cd /home/pi && tar xzvf - > /dev/null
```

 此外，还可以查看命令执行后的返回结果，如果是0，说明命令执行成功，否则，就是上传失败。
以下命令可以用于查看上一条命令的执行结果。

```bash
echo $?
```
